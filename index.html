Project Details

This project was created by Rory Turnbull for his portfolio in 2022.
Rory created this news sorting website as part of the Web Develpment and UX tertiary course run by Yoobee in Wellington for UP Education.
This is an example one page application to show how Rory has developed techniues ans skills in using external API's to source, filter and dispaly information.
Also, this one page application makes use of the Node platform and NPM packages (for example, Gulp, which transfers SCSS data into CSS) and is connected to a Github repositry.

Thank you for looking at this application, please visit Rory's Github page or use the contact details below.
